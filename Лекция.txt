                    "Команды подключения к удалённым репозиториям"
1) [Создать репозиторий] на GitHub
2) [Создать папку с проектом] на локальном компьютере или войти в нее.
    Создадим папку с именем: <Get> 
3) Войти в терминал и перейти в папку с проектом: [cd "C:\Users\Валерий Крылов\Desktop\GB\СПЕЦИАЛИЗАЦИЯ DevOps\9. Контроль версий углублённо (GIT)\Урок 1. Работа с удалёнными репозиториями"]
4) [git init] : инициализируем в этой папке репозиторий
5) [git add .\Get.html] : добавить файл в удаленный репозиторий
6) [git commit -m 'initial Get' ] : составить коммит для отправки в удаленный репозиторий
7) [git remote add origin git@github.com:Markovka28/GetWork.git] 
    подключить репозиторий к удаленному
8) [git push -u origin master] : отправить локальный репозиторий в удаленный репозиторий
    * [-u] : текущую ветку связать с удаленной
9) обновить страницу с удаленным репозиторием

10) [git clone git@github.com:Markovka28/GetWork.git] : клонировать удаленный репозиторий в папку с локальным
11) [cd GetWork/] : перейти в папку с локальным репозиторием
